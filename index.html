Visual IP Address Aggregation Tool

How to Use

Enter IPv4 and/or IPv6 CIDR addresses in the textarea (one per line or comma-separated).

Click the Go button (or press Ctrl+Enter) to merge overlapping and adjacent ranges.

Click Copy to copy the aggregated results to your clipboard for easy sharing.

Features

  • IPv4 support - Aggregate IPv4 CIDR ranges
  • IPv6 support - Aggregate IPv6 CIDR ranges
  • Mixed mode - Process both IPv4 and IPv6 together
  • Smart sorting - IPv4 sorted first, then IPv6, each numerically
  • Diff visualization - See what changed between input and output
  • Keyboard shortcut - Press Ctrl+Enter to aggregate quickly
